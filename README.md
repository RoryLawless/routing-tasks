# Data preparation tasks for routing

## Overview
This repo contains code and files required for inputs into routing API. It
contains code to:

1. extract and makes necessary adjustments to the WMATA GTFS data
2. extract and filter the US South OpenStreetMap regional data down to WMATA
   service area
